# Archive

This directory keeps material that still matters for reference, but is no longer the main interface of the repository.

- `legacy-subagents/` contains the original category-based `awesome-codex-subagents` layout.
- `upstream/` contains imported snapshots used during the rebuild.

If you are adding new work, it should usually go in the root `skills/`, `agents/`, or `prompts/` directories instead.
